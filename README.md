# النظام المحاسبي (ويب)

مشروع ويب محاسبي متعدد المستخدمين يعتمد على **Node.js + Express + SQLite**.

## الميزات الأساسية
- تسجيل مستخدمين متعددين وتسجيل الدخول.
- دليل الحسابات (CRUD).
- القيود اليومية مع التحقق من توازن المدين والدائن.
- العملاء والموردين.
- سندات قبض وصرف.
- تقارير: ميزان المراجعة، قائمة الدخل، قائمة المركز المالي.

## التشغيل المحلي
1) انسخ ملف البيئة:

```powershell
Copy-Item .env.example .env
```

2) تثبيت الاعتمادات وتشغيل السيرفر:

```powershell
npm install
npm start
```

ثم افتح:
```
http://localhost:3000
```

## النشر على Render (تشغيل فعلي)
1) ادخل على Render وأنشئ حسابًا إذا لم يكن لديك.
2) اختر **New +** ثم **Web Service**.
3) اربط المستودع `stirring-sunflower-e1e361`.
4) Render سيقرأ `render.yaml` تلقائيًا ويضبط التشغيل.
5) اضغط **Deploy** وانتظر الرابط.

> ملاحظة: قاعدة البيانات SQLite ستُنشأ تلقائيًا على السيرفر.

## النشر على GitHub Pages (نسخة Static)
هذه النسخة تعمل بدون سيرفر وتستخدم التخزين المحلي في المتصفح.

1) فعّل GitHub Pages على الفرع `main` والمجلد `/ (root)`.
2) افتح الرابط:
```
https://ahmedhaysam53-commits.github.io/stirring-sunflower-e1e361/
```

## ملاحظات
- قاعدة البيانات تُنشأ تلقائيًا عند التشغيل.
- لتغيير منفذ التشغيل عدّل `PORT` في `.env`.
